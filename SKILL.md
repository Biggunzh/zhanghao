# 月报自动化 Skill

## 功能说明

自动根据原始Excel数据填充Word月报模板，实现数据整理和清洗自动化。

## 目录结构

```
D:\月报自动化
├── 月报原始数据/          # 原始Excel数据文件
│   ├── 2026-03月报资源使用率详情列表.xls
│   ├── 2026-03工单总量.xlsx
│   ├── 2026-03-堡垒机.xlsx
│   └── 2026-03vpn审计.xlsx
├── 月报模板/              # Word模板文件
│   └── 政务云服务运维月报-2025年11月-北京市农林科学院.docx
├── 输出月报/              # 生成的月报（自动创建）
└── 月报自动化.py          # 主脚本
```

## 安装依赖

```bash
pip install openpyxl python-docx xlrd
```

## 使用方式

### 方式一：单个模板处理

```python
from 月报自动化 import generate_monthly_report

# 处理单个模板
template_path = r'D:\月报自动化\月报模板\政务云服务运维月报-2025年11月-北京市农林科学院.docx'
output_path = r'D:\月报自动化\输出月报\政务云服务运维月报-2026年03月-北京市农林科学院.docx'

generate_monthly_report(template_path, output_path)
```

### 方式二：批量处理

```bash
cd D:\月报自动化
python 月报自动化.py
```

## 工作流程

1. **读取原始数据**
   - 解析资源使用率详情列表 → 获取业务系统和主机数据
   - 解析工单总量 → 获取技术支撑统计数据
   - 解析堡垒机日志 → 获取安全审计数据
   - 解析VPN审计 → 获取VPN访问记录

2. **提取业务系统**
   - 从Word模板中识别业务系统名称（如"微营销"、"长城网"）
   - 作为数据匹配的索引键

3. **数据匹配与计算**
   - 根据业务系统名称筛选原始数据
   - 计算：主机数量、CPU总量、内存总量、存储总量
   - 计算：平均使用率、工单数量

4. **填充Word模板**
   - 替换日期（上个月）
   - 更新基础资源台账概况段落
   - 更新各章节数据表格
   - 更新附件表格

5. **生成输出文件**
   - 保持原始格式不变
   - 仅替换数据内容

## 配置参数

在脚本中可配置的参数：

```python
BASE_DIR = r'D:\月报自动化'                    # 基础目录
RAW_DATA_DIR = os.path.join(BASE_DIR, '月报原始数据')  # 原始数据目录
TEMPLATE_DIR = os.path.join(BASE_DIR, '月报模板')      # 模板目录
OUTPUT_DIR = os.path.join(BASE_DIR, '输出月报')        # 输出目录
```

## 注意事项

1. **数据文件命名**：保持原始数据文件命名格式一致（年月格式）
2. **模板格式**：不要修改Word模板的表格结构和样式
3. **日期替换**：自动将模板中的"2025年11月"替换为"上个月"
4. **编码问题**：确保Python脚本使用UTF-8编码
